# Unix-Command

## List
```
ls
```
## Print working directory
```
pwd
```
## Change directory
```
cd
cd .. // move 1 level up
cd ~ // root directory
```
## Shortcut
1. Ctrl A (Move to the front)
2. Ctrl E Move to the end)
3. Ctrl U (Clear the whole line)
## Make directory
```
mkdir Angela
```
## Create new file
```
touch Text.txt
```
## Open file
```
start Text.txt
code Text.txt
```
## Delete file
```
rm Text.txt
```
## Delete all files
```
rm *
```
## Delete folder
```
rm -r Angela
```
